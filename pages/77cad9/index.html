基础系统安装

# 介绍

基础系统安装

这里选择的是centos7的minimal镜像，仅包含基础的系统软件，更轻量级，其他软件可按需安装。

# 硬件清单

  • 个人电脑（用来烧制镜像）
  • 占美小主机一台
  • 键盘
  • 鼠标
  • 显示器
  • U盘一个（4G以上）

# 软件清单

  • 系统镜像（CentOS-7-x86_64-Minimal-1810.iso (opens new window)）
  • 镜像烧制软件（etcher (opens new window)）

# 镜像烧制

# 安装etcher

etcher是一款开源的镜像制作工具，支持多种镜像格式

https://www.balena.io/etcher/

# 制作启动盘

通过etcher将系统镜像烧制到U盘中

  1. 插入U盘
  2. 打开安装好的etcher
  3. 点击Select image选择待烧制的镜像系统CentOS-7-x86_64-Minimal-1810.iso
  4. 点击Select drive选择目标U盘
  5. 点击Flash!执行镜像烧制
  6. 烧制完成后，弹出U盘即可

# 主机系统安装

# 组装好目标主机

  1. 确保能正常开机
  2. 确保键盘能正常使用
  3. 确保显示器正常工作
  4. 确保鼠标使用正常

# 启动盘引导

  1. 插上U盘，按下开机按钮，持续按下键盘的F12键，等待显示器显示安装界面
  2. 选择 Install CentOS 7，回车，等待进入安装界面
  3. 选择语言，这里我选择的简体中文，点击继续
  4. 设置安装信息摘要，这里仅需要注意系统安装位置即可，如果你对linux分区比较熟悉，可以选择我要配置分区，不熟悉的话选择自动配置分区即可，设置好后，点击完成按钮，回到安装信息摘要界面
  5. 点击开始安装进入配置界面，此时会执行上一步操作配置的分区设置之类的工作，在配置页面，你需要配置root密码，创建用户是可选的，这里我们先不创建用户了，待进度条读满之后，系统就安装好了
  6. 移除U盘，点击重启，便会重新启动，进入系统
  7. 当看到如下界面时，便可以使用root用户登录系统了
